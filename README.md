# Recommend-system
Douban Movies Recommendation based on FM \ NeuralFM \ LFM \ SVD++ and Tensorflow. This is a project in Social Network Analysis@FDU. 

利用豆瓣电影数据进行社交网络分析和推荐系统搭建的课程Project。数据可视化demo可以进入4_Visualization文件夹查看。

## 环境要求
Tensorflow > r1.0
Python 3.6
PyEcharts
Gephi
